int POS_Raster_DownloadAndPrintBmp(

int hPort,
 
int nPortType,

char *pszPath,

int nOrgx

)



描述

下载位图到打印缓冲区中。



参数

hPort
 
[in] 打开的端口句柄。
 
 
nPortType
 
[in] 端口类型,取值参考端口类型说明


pszPath

[in] 指向以null 结尾的包含位图文件路径及其名称的字符串。


nOrgx

[in] 指定将要打印的位图和左边界的距离。

可以为 0 到 112 mm。


返回值

如果函数成功，则返回值为 POS_SUCCESS。

如果函数失败，则返回值为以下值之一：

POS_FAIL

POS_ERROR_INVALID_HANDLE

POS_ERROR_INVALID_PATH

POS_ERROR_INVALID_PARAMETER

POS_ERROR_NOT_BITMAP

POS_ERROR_NOT_MONO_BITMAP



备注

1．位图必须是单色的。

2．在标准模式（行模式）下有效。

3．设置的位图边距与位图的宽度之和不要超过打印机的最大可打印宽度。

4．另请参考 POS_PreDownloadBmpToRAM，POS_PreDownloadBmpsToFlash，POS_S_DownloadAndPrintBmp， POS_SetLineSpacing。
