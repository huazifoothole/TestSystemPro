int POS_ReadFile(

int hPort,
 
int nPortType,

char *pszData,

int nBytesToRead,

int nTimeouts

)



描述

从串口或USB端口读数据到指定的缓冲区。



参数

hPort
 
[in] 打开的端口句柄。
 
 
nPortType
 
[in] 端口类型,取值参考端口类型说明


pszData

[in] 指向将要读取的数据缓冲区。


nBytesToWrite

[in] 指定将要读取的数据的字节数。



返回值

如果函数成功，则返回值为 POS_SUCCESS。

如果函数失败，则返回值为以下值之一：

POS_FAIL

POS_ERROR_INVALID_HANDLE

POS_ERROR_INVALID_PARAMETER



备注

1．此函数仅用来调试，部分Linux内核在使用USB接口进行读操作时需要读取偶数次。

2．此函数不支持网络接口。

3．另请参考 POS_WriteFile。
