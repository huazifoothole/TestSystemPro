int POS_SetMotionUnit(

int hPort,
 
int nPortType,

int nHorizontalMU,

int nVerticalMU

)



描述

设置打印机的移动单位。



参数

 hPort
 
[in] 打开的端口句柄。
 
 
nPortType
 
[in] 端口类型,取值参考端口类型说明


nHorizontalMU

[in] 把水平方向上的移动单位设置为 25.4 / nHorizontalMU 毫米。

可以为0到255。


nVerticalMU

[in] 把垂直方向上的移动单位设置为 25.4 / nVerticalMU 毫米。

可以为0到255。




返回值

如果函数成功，则返回值为 POS_SUCCESS。

如果函数失败，则返回值为以下值之一：

POS_FAIL

POS_ERROR_INVALID_HANDLE

POS_ERROR_INVALID_PARAMETER

POS_ERROR_FILE



备注

另请参考 POS_SetMode。
