int POS_SetTimeOuts(

int hPort,
 
int nPortType,

int nReadTimeOuts,

int nReadTimeOuts

)



描述

设置USB通讯超时时间。



参数

hPort
 
[in] 打开的端口句柄。
 
 
nPortType
 
[in] 端口类型,取值参考端口类型说明


nReadTimeOuts

[in] 指定读超时时间，单位ms。

可以为 0 到 1500000。


nWriteTimeOuts

[in] 指定写超时时间，单位ms。

可以为 0 到 1500000。



返回值

如果函数成功，则返回值为 POS_SUCCESS。

如果函数失败，则返回值为以下值之一：

POS_FAIL

POS_ERROR_INVALID_HANDLE

POS_ERROR_INVALID_PARAMETER



备注

无。


相关指令

无。
