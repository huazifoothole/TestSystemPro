int POS_WriteFile(

int hPort,
 
int nPortType,

char *pszData,

int nBytesToWrite

)



描述

发送数据到端口或文件。



参数

hPort
 
[in] 打开的端口句柄。
 
 
nPortType
 
[in] 端口类型,取值参考端口类型说明


pszData

[in] 指向将要发送的数据缓冲区。


nBytesToWrite

[in] 指定将要发送的数据的字节数。



返回值

如果函数成功，则返回值为 POS_SUCCESS。

如果函数失败，则返回值为以下值之一：

POS_FAIL

POS_ERROR_INVALID_HANDLE

POS_ERROR_INVALID_PARAMETER

POS_ERROR_FILE



备注

1．此函数仅用来调试。

2．另请参考 POS_ReadFile。
